图的定义

一个无向图 GGG 是一个有序三元组 G=(V(G),E(G),ψG)G=(V(G), E(G), \psi _G)G=(V(G),E(G),ψG​)，其中：

  • V(G)≠∅V(G) \neq \varnothingV(G)​=∅ 是顶点集合，任给 v∈V(G)v\in V(G)v∈V(G) 称为一个顶点。
  • E(G)E(G)E(G) 是边集合，任给 e∈E(G)e\in E(G)e∈E(G) 称为一条边。
  • ψG:E(G)→{{u,v}∣u,v∈V(G)}\psi_G:E(G)\rightarrow\{\{u, v\}\mid u, v\in V(G)\}ψG​:E(G)→{{u,v}∣u,v∈V(G)} 称为边与顶点之间的关联函数。

为了方便，将 E(G)E(G)E(G) 中的每条边写成一个顶点对 e={u,v}e=\{u, v\}e={u,v} 或 e=uve=uve=uv，图 GGG 可简单表示成 G=(V(G),E(G))G = (V(G), E(G))G=(V(G),E(G))。

给定图 GGG，顶点的个数 ∣V(G)∣|V(G)|∣V(G)∣ 称为图 GGG 的阶，记为 ν(G)\nu(G)ν(G)。图 GGG 的边数记为 ε(G)\varepsilon(G)ε(G)。

若 ν(G)+ε(G)\nu(G)+\varepsilon(G)ν(G)+ε(G) 为无穷大，则称 GGG 为无限图，否则称 GGG 是有限图。

在图 GGG 中，若存在边 e={u,v}e = \{u, v\}e={u,v}，则称边 eee 与两个顶点 u,vu, vu,v 关联，也称 u,vu, vu,v 是 eee 的端点，同时称 u,vu, vu,v 相邻，或称 uuu 和 vvv 是邻顶。

若两条边关联同一个顶点，则称这两条边 相邻。若 e=f={u,v}e = f = \{u, v\}e=f={u,v}，则称 e=fe = fe=f 是重边。若 e={u,u}e =\{u, u\}e={u,u}，则称边 eee 是环。

无环，无重边的图叫做简单图。

  • 完全图： nnn 个顶点的完全图 KnK_nKn​ 是一个简单图，其中任意两个顶点都相邻。
  • 二分图：二分图 GGG 的顶点集合可划分为 V(G)=X∪YV(G) = X\cup YV(G)=X∪Y，其中 X≠∅,Y≠∅X\neq \varnothing,Y\neq \varnothingX​=∅,Y​=∅ 且 X∩Y=∅X\cap Y = \varnothingX∩Y=∅。使得任意两个相邻顶点必定一个在集合 XXX 中，另一个在集合 YYY 中。完全二分图 GGG 是一个简单二分图，满足 XXX 中的任意一个顶点与 YYY 中的任意一个顶点相邻，记作 K∣X∣,∣Y∣K_{|X|,|Y|}K∣X∣,∣Y∣​。
  • 星图：星图是一个完全二分图 K1,nK_{1, n}K1,n​ 或 Kn,1K_{n, 1}Kn,1​。
  • 零图：图中仅有顶点，没有边（E=∅E = \varnothingE=∅）。

deg⁡(v)=d1(v)+2∗l(v)\deg(v) = d_1(v) + 2 * l(v) deg(v)=d1​(v)+2∗l(v)

给定图 G=(V(G),E(G))G = (V(G), E(G))G=(V(G),E(G)) 和图 H=(V(H),E(H))H=(V(H), E(H))H=(V(H),E(H))，若 V(H)⊆V(G)V(H) \subseteq V(G)V(H)⊆V(G) 且 E(H)⊆E(G)E(H)\subseteq E(G)E(H)⊆E(G)，则称 HHH 是 GGG 的一个子图，记作 H⊆GH\subseteq GH⊆G。

注意：HHH 是 GGG 的子图首先需要保证 HHH 是一个图，只满足第二个条件不能说明子图关系。

路径与连通

给定简单图 G=(V(G),E(G))G = (V(G), E(G))G=(V(G),E(G))，给出以下定义：

  • 路径：图 GGG 的一条路径 WWW 定义为 W=v0e1v1e2v2⋯ekvkW = v_0e_1v_1e_2v_2\cdots e_kv_kW=v0​e1​v1​e2​v2​⋯ek​vk​。其中 eie_iei​ 与 vi,vi+1v_i, v_{i + 1}vi​,vi+1​ 关联。称 kkk 为路径 WWW 的长度，v0v_0v0​ 与 vkv_kvk​ 分别为 WWW 的起点与终点。
  • 行迹：边不重复的路径称为行迹。
  • 轨道：顶点不重复的路径称为轨道。
  • 回路：起点与终点相同的路径称为回路。
  • 圈：除了起点与终点，没有相同顶点的回路称为圈。

若顶点 uuu 与 vvv 之间存在路径，则称 uuu 与 vvv 连通，否则称 uuu 与 vvv 不连通。

若 uuu 与 vvv 连通，将 uuu 与 vvv 之间最短轨道的长度称为 uuu 与 vvv 的距离，记为 dist(u,v)\text{dist}(u, v)dist(u,v)；若 uuu 与 vvv 不连通，则记 dist(u,v)=∞\text{dist}(u, v) = \inftydist(u,v)=∞。

若图 GGG 的任意两个顶点都连通，则称 GGG 为连通图。
